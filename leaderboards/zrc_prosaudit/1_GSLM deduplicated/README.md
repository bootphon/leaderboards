# zrc_prosaudit | GSLM deduplicated

Entry for the zrc_prosaudit, by Maureen de Seyssel, Andrea Santos, Marvin Lavechin, Hadrien Titeux, Gwendal Virlet, Arthur Thomas, Guillaume Wisniewski, Bogdan Ludusan, Emmanuel Dupoux.


## Description

HuBert + Kmeans (deduplicated) + Transformer. 
Refer to de Seyssel et al (2023) for more details.
Model used from : K. Lakhotia, E. Kharitonov, W.-N. Hsu, Y. Adi, A. Polyak, B. Bolte, T.-A. Nguyen, J. Copet, A. Baevski, A. Mohamed et al., 
“On generative spoken language modeling from raw audio,” Transactions of the Association for Computational Linguistics, vol. 9, pp. 1336–1354, 2021
