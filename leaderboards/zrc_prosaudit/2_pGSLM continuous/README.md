# zrc_prosaudit | pGSLM continuous

Entry for the zrc_prosaudit, by Maureen de Seyssel, Andrea Santos, Marvin Lavechin, Hadrien Titeux, Gwendal Virlet, Arthur Thomas, Guillaume Wisniewski, Bogdan Ludusan, Emmanuel Dupoux.


## Description

HuBert + Kmeans + MS-TLM (unit + f0 + duration) with continuous prosody. Pseudo-probability on unit only. 
Refer to de Seyssel et al (2023) for more details. 
Model used from : E. Kharitonov, A. Lee, A. Polyak, Y. Adi, J. Copet, K. Lakhotia, T. A. Nguyen, M. Riviere, A. Mohamed, E. Dupoux et al., 
“Text-free prosody-aware generative spoken language modeling,”in Proceedings of the 60th Annual Meeting of the Association for Computational Linguistics (Volume 1: Long Papers), 2022, pp. 8666–8681
